Install the Appworks Command Line Tool

Use the appworks cli to build and package your appworks apps.

Make sure you have nodejs installed and enter the following into your terminal:

npm install -g appworks

This will install the appworks command line tool. With this, you may create, modify, and package your apps.

Run the following to see all of the available options:

appworks --help

Including AppworksJS and its dependencies

Appworks JS is automatically injected into every appworks apps before being deployed to the iOS or android client. You do not need to worry about including the library yourself.

However, you must add an include tag for cordova.js and appworks.min.js in the head section of your index.html to get access to the features of appworksJS. Note that they will be a 404 during development using a mobile browser or the iOS/android emulators. Emulation is meant to provide a stripped down version of the library for basic testing.

If you are starting from a template (see Getting Started), you don't have to do anything; these include tags are already present.
