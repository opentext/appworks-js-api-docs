AppWorksJS Secure Storage makes use of powerful encryption features built into iOS. If you wish to learn more about encryption on iOS, you can read Apple's security guide here.

Storing a File

Store a file securely on the device.

    appworks.storage.storeFile => (filename, downloadUrl, success, error, options)

    filename: the name to store the file on the device
    downloadUrl: the url to download the file from
    success: a success handler to run when the execution completes successfully
    error: an error handler to run when the execution encounters an error
    options: an options object to set headers and params on the request

Retrieving a File

Retrieve a file that has been stored on the device.

    appworks.storage.getFile => (filename, success, error, options)

    filename: the name of the stored file
    success: a success handler to run when the execution completes successfully. the base 64 representation of the file is passed to this callback.
    error: an error handler to run when the execution encounters an error
    options: an options object to set the type of storage (temporary or persistent)

Uploading a File

Upload a file on the device to a remote location

    appworks.storage.upload => (filename, uploadUrl, success, error, options)

    filename: the name of the stored file
    uploadUrl:the url of the server to receive the file
    success: a success handler to run when the execution completes successfully.
    error: an error handler to run when the execution encounters an error
    options: an options object to set headers and params on the request
