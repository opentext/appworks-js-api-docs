AppWorksJS gives you access to device features, as well as information about the device.

Battery Status

    
        appworks.battery

Please refer to Cordova Documentation for usage examples

Camera

    
        appworks.camera.takePicture => (callback, error, options)

        - use the device camera to take a photo
        - callback is passed base 64 encoded representation of photo

        appworks.camera.chooseFromGallery => (callback, error, options)

        - open the device gallery to choose a photo
        - callback is passed base 64 encoded representation of photo
