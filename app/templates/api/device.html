Camera

    
        appworks.camera.takePicture => (callback, error, options)

        - use the device camera to take a photo
        - callback is passed base 64 encoded representation of photo

        appworks.camera.chooseFromGallery => (callback, error, options)

        - open the device gallery to choose a photo

Geolocation

AppworksJS gives you access to the GPS functionality of your device.

    
        appworks.geolocation.getCurrentPosition => (geolocationSuccess, error, options)

        - use the device GPS to get current location coordinates
        - geolocationSuccess callback is passed Position object
        - error callback is passed error object with status code and message

        appworks.geolocation.watchPosition => (geolocationSuccess, error, options)
        returns => String

        - watches for changes to the device's current position
        - returns an id that references the watch position interval. this id should be used with clearWatch() to
        stop watching for changes in position

        appworks.geolocation.clearWatch => (id)
        - stop watching for changes to the device's location referenced by the id parameter
