Getting Started

Appworks Command Line Interface

Use the appworks cli to build and package your appworks apps.

Make sure you have nodejs installed and enter the following into your terminal:

npm install -g appworks

This will install the appworks command line tool. With this, you may create, modify, and package your apps.

Clone a sample application

Download a starter template and get started:

appworks start --name myapp

This will install the appworks starter template, which gives you access to many of the powerful features only available through appworks.

Emulate the app

Emulate the app using the iOS simulator (Mac OSX only). Note: this will only give you access to a limited number of features. To get all of the features you must deploy your app to the appworks gateway.

cd myapp appworks emulate

Package your app

This will take the contents of your app and create a zip file in the package directory. The appworks cli will look inside of a folder named www, so make sure all of the styles, js, and html contents are placed there. You may also add an icon by placing an image named icon.png in the root of your app. Feel free to modify app.properties to change the display name or other details about your app before deploying it to the gateway.

appworks package

Upload this zip to your gateway and run the app using the appworks iOS or android client.
