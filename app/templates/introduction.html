What is AppWorksJS?

AppWorks JS is a javascript library made to go hand in hand with your appworks apps. It gives you access to things like on device tech, secure storage capabilities, offline access, and more, and it comes prepackaged with every appworks app. This guide will give you a starting point in using the AppWorksJS library in your mobile enterprise apps.

Getting Started

Use the appworks cli to build and package your appworks apps.

Make sure you have nodejs installed and enter the following into your terminal:

npm install -g cordova ionic appworks

This will install the appworks command line tool. With this, you may create, modify, and package your apps.

Download a starter template and get started:

appworks start --name myapp

This will install the appworks starter template, which gives you access to many of the powerful features only available through appworks.
